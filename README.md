## English

### 📝 Description

Personal portfolio of Laura Saldarriaga Higuita, Machine Learning Developer with a Master's degree in Engineering and a Bachelor's degree in Electronics Engineering. This website showcases my professional experience, technical skills, and featured projects in the field of Machine Learning and Data Science.

### 🌟 Features

- ✨ Modern and responsive design
- 🌓 Light/dark mode with preference persistence
- 📱 Fully mobile-responsive
- 🎨 Smooth animations and transitions
- ♿ Optimized accessibility with aria labels
- 🚀 SEO optimized with Open Graph and Twitter Cards meta tags

### 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Bootstrap 4.3.1
- Bootstrap Icons
